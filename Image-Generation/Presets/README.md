# Community Presets
## Contribution Guidelines
1. All contributions must be SFW. (Absolutely no nudity.)
2. Presets must be compatible with SwarmUI.
3. Presets must accompany 'reuse parameters' function image thumbnail.
4. Preset thumbnails must be 1:1 images.
